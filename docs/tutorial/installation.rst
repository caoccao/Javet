==================
Javet Installation
==================

Maven
=====

.. code-block:: xml

    <dependency>
        <groupId>com.caoccao.javet</groupId>
        <artifactId>javet</artifactId>
        <version>0.8.8</version>
    </dependency>

Gradle Kotlin DSL
=================

.. code-block:: kotlin

    implementation("com.caoccao.javet:javet:0.8.8")

Gradle Groovy DSL
=================

.. code-block:: groovy

    implementation 'com.caoccao.javet:javet:0.8.8'
